# tunneler
Tunnel connections through HTTP like in SSH

## Install

```bash
go get github.com/rsrdesarrollo/tunneler
cd $(go env GOPATH)/src/github.com/rsrdesarrollo/tunneler
make
cd $(go env GOPATH)/bin
```
